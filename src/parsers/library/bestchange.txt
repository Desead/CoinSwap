# Вы можете использовать наши курсы с учетом указания ссылок на наш проект.
#
# API для получения курсов у нас реализовано в виде zip-архива с несколькими текстовыми файлами внутри - http://api.bestchange.ru/info.zip
# Архив обновляется постоянно вместе с сайтом и содержит сразу все курсы обмена. Загружать его можно без ограничений. Файл обновляется каждые 5 секунд.
#
# Как распарсить:
#
# bm_cy.dat - список валют в формате:
# ID валюты; xx; название валюты; xx; xx; xx; xx
#
# bm_exch.dat - список обменников в формате:
# ID обменника; название обменника; xx; xx; xx
#
# bm_rates.dat - список курсов обмена в формате:
# ID отдаваемой валюты; ID получаемой валюты; ID обменника; курс обмена (отдать); курс обмена (получить); резерв получаемой валюты; отзывы; xx
#
# Проверяйте полученные данные на корректность, иначе из-за сбоя могут образоваться убыточные курсы.
#
# Имеется готовый код для парсинга на языке PHP, вы можете найти его во вложении.
